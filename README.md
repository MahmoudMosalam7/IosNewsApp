# 📰 iOS News App

A modern iOS application built using Swift that displays the latest news from different countries, allows users to view details, and save articles locally using Core Data.

---

## 🚀 Features

- 🌍 Browse news by country (e.g., US, Egypt)
- 🏠 Home screen with a list of latest news
- 📄 Detailed screen for each news article
- 💾 Save news articles locally using Core Data
- 📚 View saved articles anytime

---

## 🛠️ Built With

- Swift
- UIKit (Storyboard + Programmatic UI)
- Core Data
- MVVM Architecture

---

## 📱 Screens

### 1. Home Screen
Displays all news articles and allows filtering by country.

### 2. Details Screen
Shows full details of the selected news and allows saving it.

### 3. Saved News Screen
Displays saved articles from Core Data.

## ⚙️ Project Info
